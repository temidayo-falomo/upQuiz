
# UpQuiz

A Quiz App made with React Native & Expo CLI.
## Features

- Unlimited Trivia Questions
- Save Questions
- Show Stats
- Change Difficulty



## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
